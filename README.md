# Microservices E-Commerce Backend

**Tech Stack:** Spring Boot, Keycloak, OAuth2, Docker  

A modular **microservices-based e-commerce backend** designed to handle authentication, product management, and secure data flow. Built with Java Spring Boot and secured using OAuth2 with Keycloak. Each service is containerized using Docker to simulate real-world deployment.

## Features

- **Microservices Architecture:** Modular services for scalability and maintainability.  
- **Secure Authentication:** OAuth2 authentication implemented with Keycloak.  
- **API Communication:** RESTful APIs with inter-service communication.  
- **Containerization:** Services packaged with Docker for easy deployment and isolation.  

## Getting Started

### Prerequisites

- Java 17+  
- Docker & Docker Compose  
- Maven
